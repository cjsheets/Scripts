# Conky

This is my personal conky configuration. Monitors include:

* System uptime/load
* CPU usage
* Top CPU processes
* RAM usage
* Top RAM processes
* Disk I/O
* Swap usage
* Disk / Mountpoint usage
* Network usage
* Active connections
* Google Calendar
